Package sim contains the simulation logic. clsBWMain and clsBWMainWithUI are specializations of the corresponding MASON classes sim.engine.SimState and sim.display.GUIState. The other classes are providing addtional GUI functionalities (clsBWFastEntityAdapter and clsBWScenarioSelectorUI).

Important: Although clsBWMain contains a static main method it is not possible to run the ARSIN World simulation without a GUI. The MASON Physics 2D engine requires the display functionalities.

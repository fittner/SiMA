This is the implementation of the model version v38a of the psychoanalytically inspired decision unit developed by project ARS.

The main package contains the processor that executes all modules of the functional model. All other funtionality, including the modules, is located in the following set of subpackages:

  • interfaces - interfaces for the inspectors and the communication channels between the modules respectively between the modules and the knowledgebase.
  • logger - action and data logger.
  • memorymgmt - knowledgebase implementation.
  • modules - the functional modules.
  • storage - temporary storage containers that enable to exchange data between modules independently of the process cycle.
  • symbolization - methods that convert the incoming information into processable symbols.
  • tools - various tools for functional modules.
